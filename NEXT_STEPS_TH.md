# งานถัดไป

1. เปิดทดสอบหน้า `http://127.0.0.1:5173/admin` แล้วลองสร้างหมวดหมู่จริง 2-3 หมวด
2. ทดสอบเพิ่ม แก้ไข ลบ code รายชิ้นในหลังบ้าน และทดสอบนำเข้าแบบหลายบรรทัด
3. เช็กว่าหน้าร้านและ flow ซื้อสินค้าแสดงผลตามหมวดหมู่ที่สร้างไว้ได้ชัดเจน

5. เริ่มเชื่อม provider จริงตัวแรก เช่น `Wepay` หรือ `24Payseller`
6. หลังจบแต่ละรอบให้อัปเดต `PROGRESS_TH.md` และ `HANDOFF_TH.md` ก่อน commit/push
