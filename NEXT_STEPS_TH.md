# งานถัดไป

1. เปิด `http://127.0.0.1:5173/` แล้วทดสอบเมนูหมวดหมู่หน้าร้านว่ากรองสินค้าได้ตรงตามที่ต้องการ
2. เปิด `http://127.0.0.1:5173/topup` แล้วทดสอบสร้าง payment intent และกดจำลองชำระเงินบน localhost
3. ถ้าต้องการแยกการขายให้ลึกขึ้น ให้เพิ่มหน้า category list แยกตาม slug
4. ปรับหน้า `ProductPage` ให้แสดง breadcrumb หรือชื่อหมวดที่สัมพันธ์กับเมนูหน้าร้าน
5. เริ่มเชื่อม provider จริงตัวแรก เช่น `Wepay` หรือ `24Payseller`
